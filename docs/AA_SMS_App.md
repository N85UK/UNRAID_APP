# AA SMS App — Andrews & Arnold integration

AA SMS App is a self-hosted web application for UNRAID that provides complete SMS messaging capabilities using the Andrews & Arnold (AA) SMS_API. Send and receive SMS messages, manage multiple accounts, and maintain contacts — all from your UNRAID server.

## Features (summary)
- Multi-account management and secure credential storage
- Send and receive SMS with thread view and delivery status
- Built-in contact management, search and import/export
- Webhook support for incoming messages and secure tokens
- Modern UI with responsive design

Requirements: UNRAID + appdata storage; see full docs in the main repository for setup and API details.
