# AA SMS App - Andrews & Arnold SMS API Integration

**Complete SMS Management for UNRAID**

## Overview

AA SMS App is a self-hosted web application for UNRAID that provides complete SMS messaging capabilities using the Andrews & Arnold (AA) SMS_API. Send and receive SMS messages, manage multiple accounts, and maintain contacts - all from your UNRAID server.
